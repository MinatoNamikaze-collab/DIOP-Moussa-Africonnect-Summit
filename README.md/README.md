# AfriConnect Summit 2026

## Projet d'Examen - Technologies Web

### Auteur
**VOTRE NOM PRENOM** - L1 Informatique

### Description
Site vitrine complet pour la conférence tech panafricaine fictive "AfriConnect Summit 2026".
L'événement réunit développeurs, entrepreneurs et investisseurs du continent pour 3 jours d'innovation.

### Technologies Utilisées
- **HTML5** - Structure sémantique (header, nav, main, section, article, footer)
- **CSS3** - Flexbox, Grid, Variables CSS, Animations, Transitions, Media Queries
- **JavaScript Vanilla** - DOM, Événements, IntersectionObserver, localStorage
- **Git & GitHub** - Versionning et déploiement GitHub Pages

### Fonctionnalités JavaScript Implémentées
1. ✅ Dark/Light Mode avec persistance localStorage
2. ✅ Navbar dynamique avec changement de fond au scroll
3. ✅ Menu hamburger responsive
4. ✅ Compte à rebours en temps réel
5. ✅ Compteurs animés au scroll
6. ✅ Animations fade-in avec IntersectionObserver
7. ✅ Onglets interactifs (Programme 3 jours)
8. ✅ Filtrage dynamique des intervenants
9. ✅ Validation de formulaire complète
10. ✅ Bouton retour en haut avec smooth scroll
11. ✅ Année dynamique dans le footer

### Pages du Site
1. **index.html** - Accueil avec Hero, Chiffres clés, Intervenants vedettes, Sponsors
2. **programme.html** - Programme avec onglets Jour 1, 2, 3
3. **intervenants.html** - Grille d'intervenants avec filtrage dynamique
4. **contact.html** - Formulaire d'inscription avec validation, FAQ accordéon, Google Maps
